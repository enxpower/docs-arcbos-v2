1. 文件定位

本文件定义 SnowBot Alpha/Beta 阶段最高层整机系统里程碑，用于统一导航、嵌入式、电控、机械、电气、测试和供应商交付节奏。

本文件不替代具体技术需求。专项技术开发应以对应文件为准：

2. 里程碑层级

解释原则：

3. Alpha · Prototype / Assist

Alpha 目标：

A1 系统架构冻结

类别内容
结构总体布局、底盘方案、除雪模块接口
嵌入式控制器、IO、通信架构
电气电池、BMS、配电架构
软件ROS2、控制框架
Gate可进入采购、加工、装配

输出：

A2 底盘与驱动系统打通

类别内容
结构底盘、轮组/履带、驱动安装
嵌入式左右驱动控制、编码器反馈
电气驱动器、电池、配电
Gate可前进、后退、转向、停车

输出：

A3 除雪作业模块完成

类别内容
结构前端安装结构、作业支撑结构
嵌入式作业机构控制
电气作业电机、供电
Gate可完成基础推雪 / 除雪动作

输出：

A4 Alpha 工程原型机完成

类别内容
整机结构、电气、嵌入式、基础软件
测试基础运行测试、基础除雪测试
Demo工程演示视频
Gate人可以安全控制机器完成基础作业

输出：

4. Beta · Task / Product Demo

Beta 目标：

B1 产品化结构重构

类别内容
结构模块化、一体化、装甲、维护接口
ID工业设计、整体产品感
Gate整机具备产品感，而非拼装原型状态

输出：

B2 定位与路线系统完成

类别内容
导航RTK、IMU、Odometry
软件Teach & Repeat
嵌入式运动执行接口
Gate可录制、保存、回放路线

输出：

B3 任务执行闭环完成

类别内容
导航路径跟踪
软件任务状态机
嵌入式速度执行、转向执行
安全障碍停车、降级逻辑
Gate固定场地内可按路线完成基础作业

输出：

说明：Beta 阶段最低障碍策略为安全停车；局部避障为可选增强能力，除非合同另行明确。

B4 Beta 展示状态完成

类别内容
结构外壳、装甲、维护结构
测试夜间测试、低温测试、雪地测试
Demo展示流程、展示视频
Gate整机达到系统演示状态

输出：

说明：B4 是整机展示状态 Gate，不应被解释为单一导航或嵌入式供应商独立责任。

B5 System Demonstration & Engineering Handover

类别内容
功能固定场地路线执行、Teach & Repeat、基础障碍停车、人工接管 / 急停、日志记录、产品展示状态
Gate可向投资人、合作伙伴、客户进行系统级演示，并完成工程交付

输出：
