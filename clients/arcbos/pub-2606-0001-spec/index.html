0. 系统架构决策树

0.1 充电方式——第一性原理否决地面方案

⚠️
环境否决逻辑（不可绕过） 除雪作业地面 = 积雪 + NaCl导电盐水 + 薄冰 + 砂砾。地面安装任何充电接口 = 短路 + 电化学腐蚀 + 冻融卡死。
结论：充电接口必须尾部安装，离地 ≥ 300mm。
方案功率效率充满产品化对位容差结论
地面无线1 kW87%~31 h±30 mm✗ 否决（地面积水/腐蚀/冻融）
尾部无线1 kW87%~31 h±30 mm△ 仅补电（可选）
尾部触点 Pogo Pin7.2 kW>99%~4 h±5 mm*✓ 采用

* 机械导向槽（V型尼龙）将导航 ±50mm 收敛至 ±5mm，Pogo 弹片弹性兜底

🎯
设计原则：-30°C 戴厚手套可完成，无工具，无接线操作，无裸露带电接头。
图1 — 滑入式快拆电池系统结构示意（含金手指三区分区防电弧设计）
部件规格
导轨材质硬质阳极氧化铝，抗腐蚀，低温不变形
滑入方向水平侧向（从机器人侧面），末端限位块

对位确认卡扣触发导轨末端微动开关 → BMS 硬件信号（对应互锁 C2）

分区式高电流盲插连接器（防电弧分区设计）

⚠️
术语说明：此处为工业级高电流盲插连接器，额定须覆盖 200A 充电及放电工况。非 PCB 边缘金手指。选型时须向供应商明确说明，避免提供错误产品类型。
Zone接触顺序信号内容安全逻辑
Zone 1最先接触保护地 GND + 低压辅助信号（12V/CAN）未接通 → Zone 3 物理屏蔽块阻断，无法接触
Zone 2中段BMS CAN H/L + 热管理控制线—
Zone 3最后接触48V HV+（主动力正极）/ HV-（负极）Zone 1通 → 握手 → BMS确认 → 允许闭合

断开顺序：Zone 3（HV）先断，Zone 1（GND）最后断。Zone 3 弹片缓接触（速度受控）+ 预充回路软启动，防打火。接触电阻 >5mΩ → BMS 上报 0x11，L2降额。

5.4 压铸件集成线槽

📐
所有主线束走压铸骨架内预制线槽，禁止独立敷设线缆。彻底解决振动磨损、低温变硬、走线凌乱三个问题。
线槽类型截面间距要求用途
主动力槽≥35×20 mm—48V充放电，容纳双根 25mm² 铜线
信号槽≥20×10 mm距动力槽 ≥30mmCAN/传感器，防干扰
加热控制槽≥20×10 mm—PTC控制线

槽体：压铸铝，内衬尼龙绝缘套（耐-40°C）。线缆固定：每100mm一个卡扣。弯折：R≥10×线径。穿壁密封：低温等级密封接头/EPDM密封圈/低温PUR灌封胶（禁室温型硅脂/PVC）。端子：Deutsch DT或同等级。

动力线束：电池仓→压铸线槽→尾部触点，约500mm。导体初始规格 ≥25mm²，必须按 200A 连续温升测试确认（端子温升≤30K，外皮≤额定值）；不满足则升至35mm²或双线并联。
