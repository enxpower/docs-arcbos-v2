6. System Milestones 对齐交付

本附件交付物应按附件二整机系统里程碑组织，不得自建脱离整机 Gate 的交付节奏。

整机 Gate交付重点Release Package 要求
A1 系统架构冻结架构、接口、控制器、IO、协议初版架构图、接口清单、CAN协议草案、IO点表、控制器选型
A2 底盘与驱动系统打通底盘运动执行与反馈底盘控制代码/配置、CAN日志、运行测试报告
A3 除雪作业模块完成除雪机构成品控制器联调控制器型号、接线图、参数配置、启停测试、联调报告
A4 Alpha 工程原型机完成Alpha 联合运行Alpha Test Report、Known Issues、基础运行日志
B2 定位与路线系统完成Teach & Repeat 基础能力路线录制/保存/加载/回放资料、测试数据、路线回放报告
B3 任务执行闭环完成路线执行、任务状态机、安全停车联合测试报告、障碍安全停车记录、异常停车记录
B5 System Demonstration & Engineering Handover系统演示能力与工程交付完整 Release Package、交接记录、培训记录、最终 Known Issues

说明：B1 产品化结构重构、B4 Beta 展示状态完成属于整机协同 Gate，不应被解释为单一导航或嵌入式供应商的独立交付责任，除非合同另行明确。质保金（主合同 P7，总价 20%）与交付物无独立映射，仅在系统稳定运行期满后释放。
