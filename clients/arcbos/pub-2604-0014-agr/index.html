结构设计合作合同（草案）

甲方：上海阿克博斯技术有限公司

乙方：陈超

身份证号：

联系方式：

鉴于甲方正在推进 Snowbot 项目的样机研发与产品 Demo 开发，乙方具备结构设计相关能力，双方经协商一致，就乙方以兼职 / 外包方式参与甲方项目结构设计工作事宜，订立本合同。

一、合作性质

  1. 本合同为项目外包 / 技术服务合作合同，不构成劳动关系、劳务派遣关系或全职雇佣关系。
  2. 乙方以兼职方式参与甲方项目，按照本合同约定完成相应结构设计工作并交付成果。
  3. 乙方自行安排具体工作时间与工作方式，但应按照甲方要求的阶段目标、里程碑节点和周提交机制完成交付。
  4. 甲方不因乙方口头承诺、沟通记录或非正式投入承担固定全职薪资义务。
  5. 双方确认：本合同适用于乙方以兼职 / 外包方式参与项目期间；如乙方未来决定全职加入甲方，则双方应另行签署劳动合同、试用期约定及相关管理文件，自劳动关系正式建立之日起，双方权利义务按照劳动合同体系及中国劳动法律法规执行。

二、合作项目与范围

  1. 项目名称：Snowbot V1 结构设计（Alpha + Beta）
  2. 合作范围包括但不限于：
    • Alpha 阶段原型机结构设计
    • Alpha 阶段样机加工、装配、修订与验证支撑
    • Beta 阶段产品级 Demo 结构设计
    • Beta 阶段展示样机加工、装配、修订与展示支撑
    • 底盘、行走、电池仓、前端作业模块、外壳装甲、维护接口等相关结构设计工作
  3. 乙方的工作不仅包括结构设计文件输出，还包括支撑样机加工、装配、修订、验证，并最终服务于阶段样机实物落地。
  4. 具体任务范围、阶段目标、时间节点、里程碑、交付要求，以甲方确认的《Snowbot 结构设计任务分解（Alpha + Beta）》及其后续更新版本为准。该文件作为本合同附件一，与本合同具有同等法律效力。
  5. 《每周验收 Checklist》作为本合同附件二，与本合同具有同等法律效力。

三、项目时间与阶段目标

1. 项目启动时间

项目自 2026年4月20日起启动，总周期为 8 个月，计划于 2026年12月20日前完成 Beta 投资人展示版样机。

2. Alpha 阶段

Alpha 阶段为原型机阶段，时间为 2026年4月20日至2026年8月20日。

目标为完成 Alpha 原型机样机实物的结构支撑、加工支撑、装配修订与工程验证，使机器具备“做出来、装起来、跑起来、能验证”的工程状态。

该阶段不以对外展示为目标。

Alpha 阶段不仅要求设计完成，还要求形成一台可用于工程验证的 Alpha 原型机样机实物。

Alpha 阶段主要里程碑如下：

  • A1｜整机结构总体方案完成：2026年5月10日前
  • A2｜底盘系统完成：2026年6月10日前
  • A3｜前端除雪模块安装结构完成：2026年7月10日前
  • A4｜Alpha 原型机样机实物完成：2026年8月20日前

3. Beta 阶段

Beta 阶段为产品级 Demo 阶段，时间为 2026年8月21日至2026年12月20日。

目标为在 Alpha 原型基础上完成产品化优化，形成一台可供投资人展示的 Beta 投资人展示版样机实物。

Beta 阶段结构设计必须体现以下原则：

  1. 模块化
  2. 一体化
  3. 铝合金一体化压铸方向
  4. 产品感与展示效果提升

Beta 阶段不仅要求结构优化完成，还要求完成展示样机加工、装配、修订与展示状态实现。

Beta 阶段主要里程碑如下：

  • B1｜模块化结构重构完成：2026年9月15日前
  • B2｜铝合金一体化压铸方向完成：2026年10月10日前
  • B3｜外壳 / 装甲与产品感完成：2026年11月5日前
  • B4｜维护与可服务性完成：2026年11月25日前
  • B5｜Beta 投资人展示版样机实物完成：2026年12月20日前

四、交付机制

  1. 乙方须按周提交成果。
  2. 每周至少提交一次有效成果，提交内容包括但不限于：
    • 3D 原始设计文件
    • STEP 文件
    • 2D 工程图
    • 总装图 / 分总成图
    • 结构说明文档
    • 干涉检查 / 修订记录
    • Beta 阶段所需效果图
    • 加工支撑记录
    • 装配支撑记录
    • 样机照片、视频或其他可证明样机实物推进的记录
  3. 所有提交文件必须上传至甲方指定 GitHub、云盘或其他指定位置。
  4. 仅口头说明、聊天描述、截图、未上传源文件的内容，不构成有效交付。
  5. 甲方按照附件二《每周验收 Checklist》对乙方每周提交成果进行验收，并据此判断是否计入有效提交及可结算金额。
  6. 乙方确认，项目推进以里程碑和周提交为核心，不以单纯沟通或个人主观投入作为结算依据。
  7. 有效交付不仅看设计文件本身，还看该提交是否对样机加工、装配、修订、验证或展示推进产生实际作用。

五、验收与结算

  1. 甲方对乙方每周提交成果进行验收，并按 A / B / C 三类结论进行认定：
    • A：有效提交，可计入结算；须满足文件真实提交、与任务相关、可验证、可继续推进，并对样机实物推进有实际作用。
    • B：有提交，但暂不计入结算；包括文件不完整、说明不清晰、设计深度不足、对样机推进作用不明确等情况，需补充后再确认。
    • C：无效提交，不计入结算；包括无源文件、仅口头 / 截图、与任务无关、无法证明本周真实推进、无法证明对样机实物推进有实际作用等情况。
  2. 每周经甲方确认后的有效提交，计入当月可结算金额。
  3. 每月月底由甲方汇总当月已确认成果。
  4. 甲方应于次月15日前完成上月已确认部分的结算支付。
  5. 未达到交付标准或未通过验收的内容，甲方有权不予结算。
  6. 乙方未达到里程碑时间节点且无合理说明的，甲方有权在后续结算时据实际交付情况重新评估应付金额。
  7. 阶段奖金不仅取决于设计文件完成，也取决于对应阶段样机实物是否实际落地。

六、合作费用条款

  1. 双方同意采用“兼职 / 外包 + 按成果结算”的合作方式。
  2. 乙方在合同期内不享有固定全职月薪。
  3. 费用采用“基础周费 + 关键里程碑奖金”模式：
    • 基础周费：人民币 3,000 元 / 周
    • 关键里程碑奖金：根据经甲方确认的重要阶段成果另行计入
  4. 关键里程碑奖金如下：
    • A1｜整机结构总体方案完成：人民币 5,000 元
    • A2｜底盘系统完成：人民币 8,000 元
    • A3｜前端除雪模块安装结构完成：人民币 5,000 元
    • A4｜Alpha 原型机样机实物完成：人民币 5,000 元
    • B1｜模块化结构重构完成：人民币 8,000 元
    • B2｜底盘 / 电池仓压铸方向方案完成：人民币 10,000 元
    • B3｜外壳 / 装甲与展示版整机效果完成：人民币 8,000 元
    • B4｜维护与可服务性完成：人民币 5,000 元
    • B5｜Beta 投资人展示版样机实物完成：人民币 10,000 元
  5. 上述奖金项目仅在对应成果通过甲方验收、确认为有效交付，并与对应阶段样机实物落地结果一致时计入结算。
  6. 若乙方当周未形成有效提交，则该周基础周费不计入结算。
  7. 若乙方提交为 B 类，则甲方有权待乙方补充完善并重新确认后，再决定是否计入结算。
  8. 乙方确认，在未形成连续、稳定、可验证交付之前，甲方不按全职薪资方式支付报酬。

七、知识产权与成果归属

  1. 乙方在履行本合同过程中形成的全部成果，包括但不限于：
    • 结构方案
    • 3D / 2D 设计文件
    • 工程图
    • 模块接口定义
    • 效果图
    • 设计说明
    • 修订记录
    • 与项目相关的其他文档、数据、资料

    均归甲方所有。

  2. 乙方应配合甲方完成全部设计文件、源文件及相关资料的交付。
  3. 未提交至甲方指定位置的文件，视为未完成正式交付。
  4. 未经甲方书面同意，乙方不得将本项目成果用于其他项目、向第三方披露或授权第三方使用。

八、保密义务

  1. 乙方对甲方项目相关的全部技术、商业、图纸、文档、方案、客户、融资信息负有保密义务。
  2. 未经甲方书面同意，乙方不得向任何第三方披露、传播、转移或允许第三方使用。
  3. 本条保密义务不因本合同终止而失效。

九、合作管理与替代权

  1. 乙方确认，甲方有权根据项目进展需要，引入其他结构工程师、供应商或合作方共同参与项目。
  2. 乙方不得以自己为唯一结构负责人为由，对甲方项目推进形成排他性限制。
  3. 乙方应配合甲方进行设计资料同步、接口说明和必要交接。
  4. 若乙方连续两周无有效提交，或对项目推进造成明显拖延，甲方有权重新安排合作方式、缩减合作范围或终止本合同。
  5. 乙方确认，甲方可根据里程碑推进情况对项目任务进行调整，乙方应予配合。
  6. 若乙方仅完成图纸设计而未能对阶段样机实物推进形成实质支撑，甲方有权认定对应阶段目标未完成。

十、违约与终止

  1. 若乙方存在以下任一情形，甲方有权单方终止本合同：
    • 连续两周无有效提交
    • 拒绝提交源文件
    • 交付内容明显与约定不符
    • 未经合理说明严重偏离里程碑节点
    • 仅完成设计文件而未能支撑阶段样机实物推进
    • 泄露甲方保密信息
    • 擅自将项目成果用于其他用途
  2. 若甲方终止本合同，应对已验收确认的有效成果进行结算；未验收通过的部分不予结算。
  3. 合同终止后，乙方仍应继续履行保密义务和成果交付义务。

十一、全职加入后的衔接原则

  1. 如乙方未来决定全职加入甲方，双方应在正式入职前另行签署书面劳动合同、试用期约定、岗位说明书、考核制度确认文件及其他必要文件。
  2. 自劳动关系正式建立之日起，乙方的日常管理、出勤、工资发放、社会保险、试用期管理、绩效考核、岗位职责等，均应按照劳动合同、公司规章制度及中国劳动法律法规执行。
  3. 乙方转为全职后，本合同项下形成的项目时间节点、里程碑、周推进节奏、任务分解和验收 Checklist，可转化为工作计划与绩效考核依据，但不直接等同于劳动报酬的线性扣减依据。
  4. 甲方可在合法前提下，将乙方薪酬结构设置为“固定工资 + 绩效工资 / 奖金”模式，并将阶段目标达成情况、项目推进质量、文件提交质量、团队协作情况等纳入绩效评价。
  5. 甲方对乙方进行绩效管理时，应遵守劳动法及相关法规要求，包括但不限于：
    • 不低于法定最低工资标准发放劳动报酬
    • 不以未完成单项任务为由任意克扣固定工资
    • 试用期工资、转正条件、绩效规则应事先明确
    • 规章制度与考核机制应依法建立并向乙方明示
  6. 如乙方转为全职后仍继续承担 Snowbot 结构设计工作，则甲方有权依据项目进度、里程碑完成质量、交付纪律和考核结果，对其岗位胜任情况、绩效工资、奖金发放、转正结论及后续任用进行综合评估。
  7. 乙方确认，全职加入并不当然意味着取消考核；相反，岗位要求、时间节点、交付质量和绩效结果将作为全职管理的重要组成部分。

十二、其他

  1. 本合同未尽事宜，由双方另行协商并签署补充协议。
  2. 本合同附件与补充协议与本合同具有同等法律效力。
  3. 因本合同引起的争议，双方应先友好协商；协商不成的，提交甲方所在地有管辖权的人民法院处理。
  4. 本合同自双方签字或盖章之日起生效。
