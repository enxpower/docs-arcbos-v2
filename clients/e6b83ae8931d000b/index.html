2. SnowBot整机全貌

SnowBot是一台低温户外自主除雪机器人。整机不是消费级小车，而是低温、高电流、雪水、盐雾、振动、无人值守工况下运行的商业设备。

整机电气系统

A包：电池能源包
├─ 48V LFP电池包
├─ BMS
├─ 主回路保护
└─ 低温禁充

B包：整车电控总包
├─ 除雪总成主电机与控制器
├─ 履带电机与控制器
├─ VCU
├─ IO
├─ 状态机
└─ 初版低压与线束

C包：产品化电气工程包
├─ 充电Dock
├─ 尾部DC触点
├─ 正式线束
├─ 低压配电优化
└─ EMC工程化

I包：ARCBOS总架构包
├─ 系统架构
├─ 接口控制
├─ 安全边界
├─ 集成测试
└─ 最终冻结

A包必须理解一个事实：SnowBot的高电流不是由单个负载产生，而是除雪总成、履带、低压系统、热管理在重载雪况下叠加产生。

5. 为什么A包必须按300A级设计

SnowBot不是单一电机负载。重载雪况下，除雪总成、履带和热管理会叠加。

叠加负载来源

除雪总成
├─ 正常除雪：约80–130A
├─ 重载除雪：约130–180A
└─ 峰值/堵转：可能200–300A

履带系统
├─ 左履带：重载40–70A，峰值80A以上
└─ 右履带：重载40–70A，峰值80A以上

低压与热管理
└─ 约20–50A，取决于DC-DC、PTC和控制系统

合计
├─ 中等工况：约130–240A
├─ 重载工况：约230–360A
└─ 峰值工况：可能超过400A

因此，300A连续等级不是过度设计，而是SnowBot 48V平台成立的基本条件。

8. 电池包结构设计要求

A包必须把电池包作为工业模块设计，而不是把电芯、BMS和外壳简单装在一起。电池包结构设计属于A包核心责任范围，并必须与SnowBot整机结构、电气接口、维护方式和低温环境共同设计。

电池包结构设计目标

安全
├─ 电池包未锁止不得上电
├─ 半插入不得上电
├─ 带载拔出必须被硬件阻止
├─ Service Disconnect必须可达
└─ 主回路高电流端子必须防误触

维护
├─ 电池包可抽拉或可拆装
├─ 维护人员可识别锁止状态
├─ 连接器可检查
├─ 关键保护器件可维护
└─ 电池包可安全搬运或吊装

环境
├─ 防雪
├─ 防水
├─ 防盐雾

├─ 防冰
└─ -30°C低温操作

├─ 与线束路径匹配

8.1 电池包外壳

电池包外壳必须满足以下要求：

  1. 能承受Alpha约13.44kWh电池包和Beta/v1.0约26.88kWh电池包对应重量。
  2. Beta/v1.0电池包应按约162kg以上等级进行结构设计和安全搬运评估。
  3. 外壳应具备足够刚度，避免运输、振动、抽拉、安装过程中对电芯、铜排、BMS和连接器产生应力。
  4. 外壳材料、密封件、卡扣、锁扣和把手必须适应-30°C低温。
  5. 外壳应防水、防雪、防盐雾，并具备必要排水路径。
  6. 外壳不应把冷凝水、融雪水或盐水引向高电流端子和BMS。

8.2 滑入式安装与抽拉路径

若采用滑入式电池模块，A包必须提供电池包结构接口要求，并与整机结构方确认。

└─ 低温下可操作

8.3 盲插连接器与安装面

电池盲插连接器属于A包核心交付范围。连接器必须同时考虑高电流、低压信号、互锁、半插入检测和维护安全。

要求：

  1. 高电流正负极必须防误插、防反插。
  2. 高电流触点必须满足主放电和充电路径要求。
  3. 低压信号触点应包含BMS通信、互锁、锁止状态和必要辅助信号。
  4. 盲插安装面必须有机械导向，不能让高电流端子承担定位力。
  5. 半插入状态不得上电。
  6. 未锁止状态不得上电。
  7. 带载拔出必须被硬件阻止。
  8. 连接器必须提供插拔寿命、温升、盐雾、振动和低温资料。

8.4 Service Disconnect可达性

Service Disconnect必须进入电池包结构设计，不得后期临时找位置。

要求：

  1. 维护人员可在不拆大量结构件的情况下操作。
  2. 操作位置应防误触，但不能不可达。
  3. 操作状态应清楚可见。
  4. 若Service Disconnect位于主放电路径，必须按250–300A连续等级评估。
  5. 若Service Disconnect不承载主路径，必须在原理图和结构图中明确限制。

8.5 电池仓环境接口

A包必须向整机结构方输出电池仓环境要求。

电池仓环境接口

├─ 排水路径不得经过高电流端子

├─ PTC或加热片安装空间
├─ 温度传感器位置

13. 供应商必须回答的问题

  1. 是否具备15S LFP 300A连续等级电池包设计经验。
  2. BMS是否支持300A连续、400–450A 10秒、500–600A 1秒。
  3. BMS是否支持200A连续充电。
  4. 低温禁充是否为硬件级保护。
  5. 是否能提供完整CAN协议、DBC或寄存器表。
  6. 是否能提供故障码表和保护动作日志。
  7. 主接触器、主熔断器、Service Disconnect是否有DC参数和曲线。
  8. 是否能提供I²t初步协调。
  9. 是否能提供电池包温升测试。
  10. 是否接受所有源文件、图纸、协议和测试报告交付给ARCBOS。

14. A包验收标准

A包通过验收至少满足：

  1. 设计资料完整，不只交PDF宣传资料。
  2. BMS CAN通信可被VCU读取。
  3. 低温硬件禁充可验证。
  4. 主接触器、预充、主熔断器逻辑清楚。
  5. 连续放电、峰值放电、连续充电能力有供应商资料和测试依据。
  6. 电池包接口、端子、盲插、锁止、半插入检测清楚。
  7. 故障码、日志、保护动作可追溯。
  8. 与B包、C包、I包接口清楚。

15. A包一票否决项

A包一票否决项

1. 低温禁充只靠软件
2. BMS没有CAN故障码
3. 无法证明300A连续能力
4. 无法证明400–600A峰值能力
5. 无法证明200A连续充电能力
6. 主熔断器无DC分断能力资料
7. 接触器无DC参数或无反馈策略
8. Service Disconnect低于主路径电流等级且未说明限制
9. 电池半插入仍可能上电
10. 带载拔出没有硬件阻止
11. 无温升测试计划

18. BMS CAN接口与温升验证强制要求

本节定义A包必须满足的BMS通信接口、心跳超时、温升验证和新增否决要求。供应商必须按本节提交可集成、可测试、可验收的工程资料。

18.1 BMS CAN协议最低规范

A包不得只要求供应商“提供CAN协议”。供应商必须交付可直接用于VCU集成和仿真的标准化接口文件。

BMS CAN协议最低交付要求

文件格式
├─ 必须提供标准DBC文件
├─ 必须提供可读版信号说明表
├─ 必须提供故障码表
├─ 必须提供通信超时处理说明
└─ 不接受只交PDF截图或宣传手册

CAN基础参数
├─ CAN 2.0B
├─ 11位或29位ID必须明确
├─ 波特率必须明确，候选250kbps或500kbps
├─ 周期报文刷新周期必须明确
└─ 心跳帧必须定义

VCU必须读取的核心信号
├─ PackVoltage
├─ PackCurrent
├─ SOC
├─ SOH
├─ SOP
├─ MinCellVoltage
├─ MaxCellVoltage
├─ MinCellTemperature
├─ MaxCellTemperature
├─ ChargeAllowed
├─ DischargeAllowed
├─ RegenAllowed
├─ MainContactorStatus
├─ PrechargeStatus
├─ InterlockStatus
├─ FaultLevel
└─ FaultCode

18.2 BMS CAN核心信号表

信号最低要求用途
ChargeAllowed1 bit或等效状态位，周期发送VCU和充电系统判断是否允许充电
DischargeAllowed1 bit或等效状态位，周期发送VCU判断是否允许驱动和除雪
RegenAllowed1 bit或等效状态位，周期发送VCU执行满电下坡和回生限制
PackVoltage必须定义单位、比例、偏移、有效范围主母线监控和欠压保护
PackCurrent必须定义充放电方向、单位、比例、偏移负载仲裁、保护和日志
SOC必须定义范围0–100%续航、满电下坡、低电量策略
MinCellTemperature必须可用于低温禁充判断任一电芯低于5°C禁止充电
FaultCode必须有故障码表和故障等级VCU、日志、售后诊断

18.3 心跳与超时规则

BMS通信超时最低规则

心跳帧
├─ 推荐周期：100ms以内
├─ 最大允许周期：由A包供应商说明，I包批准
└─ 必须包含BMS在线状态或递增计数

VCU超时处理
├─ BMS关键状态超过300ms未更新：VCU进入DERATE或保持安全状态
├─ BMS关键状态超过1s未更新：VCU禁止新动作
├─ BMS关键状态超过3s未更新：VCU进入FAULT_STOP或LOCAL_SAFE
└─ 具体时间可在系统集成测试后由I包冻结

以上时间为Alpha阶段建议初值。供应商如提出不同时间，必须说明理由，并经I包批准。

18.4 高电流温升测试通过标准

A包必须为温升测试提供量化通过标准，不能只写“完成温升测试”。

温升测试最低通过标准

200A连续充电路径
├─ 按200A连续运行到温度稳定
├─ 高电流连接器端子温升目标≤30K，或不超过连接器厂家额定温升
├─ 线缆外皮温度不得超过材料额定值
└─ 接触电阻不得出现异常上升

250–300A主放电路径
├─ 按设计连续电流运行到温度稳定
├─ 端子、铜排、线缆、接触器、熔断器座均需测温
├─ 任何测点不得超过器件额定温度
└─ 温升结果必须进入A包测试报告

盲插连接器
├─ 分别测试充电电流和放电电流工况
├─ 插拔后复测接触电阻
└─ 低温/盐雾/振动后复测接触电阻

18.5 A包新增一票否决项

A包新增一票否决项

1. 只提供PDF协议说明，不提供DBC文件或可集成信号表。
2. BMS没有ChargeAllowed、DischargeAllowed、RegenAllowed三个关键状态。
3. BMS通信无心跳或无超时处理说明。
4. 温升测试无量化通过标准。
5. 200A充电路径和250–300A主放电路径未分别进行温升验证。
